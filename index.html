V1 Quick Build

First Version (No Pipeline)

Scraped the site, picked fonts and colors by gut, wrote the HTML. No knowledgebase, no copy wireframe, no design system. Just vibes and Tailwind.
V2 Pipeline Build

Second Version (Full Pipeline)

Built from the company knowledgebase, sitemap strategy, copy wireframe, and design system spec. Every section follows the messaging framework. Choreographed hero animation, proper GSAP sequencing, reduced-motion support, 4s safety timeout.
DS Component Preview

Design System Preview

Living HTML component reference. Color swatches, type scale, button variants, every card type, form elements, rendered section components, icon library, animation catalog, spacing reference, and background alternation map.

Pipeline Documents (in /docs)

Step 1

Company Knowledgebase

Business identity, services, audiences, differentiators, social proof, brand direction, conversion architecture, SEO foundation.

Step 1

Sitemap Strategy

9-section homepage map with navigation structure, section types, briefs, image direction, CTA placement, and background alternation.

Step 2

Copy Wireframe

Messaging framework (voice, hierarchy, CTAs, StoryBrand map) plus section-by-section wireframes with headline options, body direction, proof points, tone calibration.

Step 3

Design System Spec

Color tokens, type scale, spacing system, icon library, component selections, section backgrounds, animation patterns (Level 2), responsive breakpoints, button and form specs.
