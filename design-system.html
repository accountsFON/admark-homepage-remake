Animation Patterns (Level 2)

Pattern Trigger Duration Behavior
fade-up ScrollTrigger (top 85%) 800ms opacity 0 > 1, translateY 40px > 0. power3.out. Once.
stagger-item ScrollTrigger (top 75%) 600ms + 100ms stagger Same as fade-up but items animate sequentially.
card-hover CSS :hover 300ms translateY(-6px) + shadow increase.
button-hover CSS :hover 300ms translateY(-2px) + teal glow shadow.
nav-scroll scrollY > 80 300ms Transparent > dark blur background.
scroll-progress Window scroll Continuous Left-edge bar fills based on scroll %.
